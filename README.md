cordova-sailfish
================

Cordova-sailfish is a SailfishOS port of the Apache Cordova project. 

#Requirements

[Saiflish OS SDK](https://sailfishos.org/)

#Supported APIs

* Accelerometer
* Battery
* Camera
* Compass
* Connection
* Device
* Events
* in-app browser
* Notification
* Vibration


# Work in progress

* Contacts
* Geolocation
* Storage
